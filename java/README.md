# Overview
A simple console-based **To-Do List application** written in Java.  
This project is designed to help practice Java such as:
- Variables
- Arrays/Lists
- Loops
- Conditionals
- Functions
- Basic Object-Oriented Programming (OOP)

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Exit the program
